# How to use:
double click BrawlCV.exe on Windows <br />
run BrawlCV.exe with Wine on Mac OS <br />
Run the script: <br />
  install python 3 <br />
  run cmd.exe <br />
  run command pip install all modules in requirements.txt <br />
  e.g. pip install Pillow  <br />
  run command 'python BrawlCV.py' <br />
